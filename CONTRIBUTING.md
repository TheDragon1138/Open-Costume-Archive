## Modelling Standards

Models should aim to:

- Match original production geometry
- Use documented references wherever possible
- Avoid speculative reconstruction where evidence is lacking
- Record assumptions within documentation
- Preserve revision history

## Research Standards

Every claim should ideally include:

- Source
- Date
- Photographer (where applicable)
- Confidence level
- Notes

Research should distinguish between:

- Confirmed
- Likely
- Speculative

## Versioning

This project follows semantic versioning.

Example:

```
v1.2.0
```

- Major = new helmet release
- Minor = significant improvements
- Patch = documentation or modelling fixes
